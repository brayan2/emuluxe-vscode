# Your New Mobile Sandbox

Gone are the days of context-switching between your editor and an un-responsive Chrome window or heavyweight Android Studio Emulator.

1. Once authenticated, run the **`Emuluxe: Start Simulation`** command via the Command Palette.
2. An interactive Quick-Pick dropdown will appear—literally choose any standard smartphone (e.g., iPhone 15 Pro Max, Galaxy S24 Ultra)!
3. Type the `localhost` URL pointing to your development sever. 

A new Editor tab spawns embedding the fully modeled 3D hardware and the Emuluxe viewport.

### Core Shortcuts (Active Tab)
- **`Cmd+R`**: Flip orientation to landscape.
- **`Cmd+S`**: Capture a high-res device screenshot including site content.
- **`Cmd+I`**: Open the web inspector.
- **`Cmd+Alt+D`**: Change the current device.
- **`Cmd+Alt+X`**: Stop the simulation and free resources.
